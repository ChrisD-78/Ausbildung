# Fachangestellte für Bäderbetriebe - Ausbildungsseiten

Diese Website bietet umfassende Informationen zur Neuordnung des Berufsbildes "Fachangestellte für Bäderbetriebe" sowie interaktive Ausbildungsplanungstools.

## 📋 Inhalte

### 1. Neuordnung des Berufsbildes
- Übersicht über die geplante Modernisierung der Ausbildung
- Hauptänderungen auf einen Blick
- Digitalisierung, Kommunikation, Bädertechnik
- Prüfungsänderungen und Zeitplan

### 2. Ausbildungsrahmenplan
- Vollständiger Rahmenlehrplan nach KMK-Beschluss vom 21. November 1996
- Alle 3 Ausbildungsjahre mit Lerngebieten
- Lernziele und Lerninhalte strukturiert dargestellt

### 3. Interaktiver Ausbildungsplan
- Ausfüllbares Formular für Ausbildungsplanung
- Tabelle mit allen Fertigkeiten und Kenntnissen
- Automatisches Speichern im Browser
- Druckfunktion

## 🚀 Verwendung

Die Seite kann lokal geöffnet werden oder über GitHub Pages veröffentlicht werden.

### Lokale Verwendung
```bash
# Mit Python
python3 -m http.server 8000

# Dann im Browser öffnen:
# http://localhost:8000
```

### GitHub Pages
1. Repository auf GitHub erstellen
2. In den Repository-Einstellungen → Pages aktivieren
3. Branch `main` als Quelle auswählen
4. Die Seite ist dann unter `https://[username].github.io/[repository-name]` erreichbar

## 📁 Dateien

- `index.html` - Hauptseite mit Neuordnungsinformationen
- `ausbildungsrahmenplan.html` - Vollständiger Ausbildungsrahmenplan
- `ausbildungsplan.html` - Interaktiver Ausbildungsplan
